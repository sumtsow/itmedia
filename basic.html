C# basics

Начало курса:
23 марта

О курсе:

Язык C# является одним из наиболее востребованных языков программирования в мире. В настоящее время C# широко используется для разработки приложений различной сложности, ориентированных на платформу .NET, которая является неотъемлемой частью Windows. Кроме того, С# является базовым языком разработки игровых приложений на основе технологии Unity3d. Данный курс ориентирован на тех, кто только начинает изучать язык С# и хотет научится использовать основные его возможности, и не имеет большого опыта в программировании. Курс C# - Basics позволит вам научиться писать простые приложения для Windows. Пройдя данный курс вы сможете:
  1. Разобраться с системой типов в C#;
  2. Изучить принципы объектно-ориентированного программирования на примере C#;
  3. Изучить основы работы с коллекциями и распространенными структурами данных;
  4. Научиться работать с делегатами и событиями;
  5. Освоить на практике работу с системами контроля версий.

Длительность курса:
3 месяца (12 занятий по 3 часа).

Преподаватель:
Фомичев А.А.
