Angular

Начало курса:
15 января

О курсе:

Курс посвящен проектированию клиентской части веб-приложений на базе фреймворка Angular. В рамках данного курса будут рассмотрены типовые задачи, с которыми сталкиваются фронтенд разработчики, и способы их решения средствами фреймворка Angular. Курс рассчитан на людей знакомых с основами алгоритмизации и принципами объектно-ориентированного программирования, и, владеющих хотя бы, одним языком программирования.

Длительность:
2 месяца (32 часа).

Стоимость:
2 700 грн/мес

Преподаватель:
Гиковатый В.М.

Содержание курса:

  • Тема 1. Введение (4 часа)
    1. Особенности Angular: история создания, назначение, версии, альтернативные фреймворки
    2. Инструменты разработки: Visual Studio Code, Node.JS, npm, webpack, git.
    3. Typescript: что такое Typescript, почему его следует использовать, основные особенности языка.
  • Тема 2. Общая архитектура Angular (4 часа)
    1. Создание первого приложения. С помощью нескольких команд Angular CLI создадим и запустим Angular приложения.
    2. Структура Angular приложения. Проведем анализ структуры, созданного проекта. Рассмотрим основные файлы и ключевые настройки, а также узнаем об основных программных блоках Angular и обсудим вопросы их организации в приложении.
    3. Модули в Angular. Вспомним, что такое модуль. Рассмотрим объявление модуля на примере главного модуля Angular приложения. Выполним установку библиотеки Angular Material и подключим несколько ее модулей. И в заключении, создадим и подключим отдельный модуль для универсальных компонентов пользовательского интерфейса.
    4. Компоненты и шаблоны. Посмотрим на компонент, как на дом для кода, обеспечивающего взаимодействие пользователя и веб-приложения, а на шаблон – как на дом для DOM (HTML разметки).
    5. Привязка данных. Увидим, как данные, становятся частью веб-страницы. Попробуем несколько техник привязки данных, предлагаемых Angular.
    6. Форматирование данных или Pipe. Обсудим, что такое форматирование данных и зачем оно нужно. Исследуем возможности стандартных pipes: установим знак доллара перед суммой ЗП, выведем дату, в формате привычном американскому заказчику и т.п.
    7. Сервисы. Рассмотрим, что такое сервис, его место в архитектуре проекта. Научимся создать сервис, регистрировать его в модуле, а также использовать в компонентах.
  • Тема 3. Компоненты (2 часа)
    1. Input свойства компонента.
    2. Output свойства компонента.
    3. Использование геттеров и сеттеров.
    4. Проекции.
    5. Жизненный цикл компонента.
    6. Инициализация компонента ngOnInit. Обработка onInit.
    7. Обработка изменений ngOnChange. Обработка onChange.
  • Тема 4. Директивы (2 часа)
    1. Директива ngStyle.
    2. Директива ngClass.
    3. Директива ngIf.
    4. Директива ngSwitch.
    5. Директива ngFor.
    6. Что означает звездочка (*) в имени директивы.
    7. Ссылочные перемененные (Template Reference Variable).
  • Тема 5. Маршрутизация (4 часа)
    1. Что такое маршрутизация.
    2. Примеры реализации маршрутизации.
    3. Относительные параметры. Передача и чтение параметров.
    4. Дочерние маршруты.
    5. Относительный путь.
    6. Guard.
  • Тема 6. Формы (4 часа)
    1. Обработка событий элементов пользовательского интерфейса.
    2. Подходы к созданию форм.
    3. Структура формы.
    4. Управление отображением элементов формы.
    5. Разметка для отображения ошибок.
    6. Валидация данных формы.
  • Тема 7. Взаимодействие с сервером по HTTP (4 часа)
    1. Виды запросов: GET, POST, PUT, DELETE.
    2. HttpClientModule и HttpClient.
    3. Создание сервиса для взаимодействия с сервером по HTTP.
    4. Конфигурирование запросов.
    5. Обработка ответов сервера.
    6. Пример загрузки файла на сервер.
  • Практика: работа над проектом (8 часов)
